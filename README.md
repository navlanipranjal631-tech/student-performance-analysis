# 📊 Student Performance Data Analysis

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a student dataset to understand the factors affecting academic performance.

---

## 🎯 Objective
To analyze student scores and identify patterns based on:
- Gender
- Lunch type
- Test preparation course
- Parental level of education

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab 

---

## 📂 Dataset
The dataset contains student performance data including:
- Math Score
- Reading Score
- Writing Score
- Demographic and academic factors

---

## 📊 Analysis Performed
- Data cleaning and inspection  
- Statistical summary  
- Distribution analysis (histograms)  
- Grouped analysis (gender, lunch, test preparation)  
- Data visualization using bar charts  

---

## 🔍 Key Insights
- Students scored higher in **reading and writing** compared to math  
- Students who completed the **test preparation course performed better**  
- Students with **standard lunch scored higher** than those with free/reduced lunch  
- Female students performed better in **reading and writing scores**

---

## 📈 Visualizations
- Math score distribution  
- Score comparison by gender  
- Impact of test preparation  
- Impact of lunch type  

---

## 🧾 Conclusion
The analysis shows that academic performance is influenced by multiple factors such as preparation, nutrition, and demographic variables. Data visualization helps in identifying trends and making informed conclusions.
